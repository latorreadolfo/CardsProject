# React Native Card Styling and Flex Practice

This repository is a practice ground for honing skills in styling cards and mastering the flex layout system in React Native.

## Features

- **Card Styling:** Experiment with different card layouts and visual styles using React Native components.
- **Flex and Flex Direction:** Gain experience in using `flex` and `flexDirection` for creating responsive designs.

## Technologies/Frameworks Used

- React Native

## Getting Started

1. **Clone the Repository:** `git clone https://github.com/latorreadolfo/CardsProject.git`
2. **Install Dependencies:** `cd CardsProject` then `npm install`
3. **Run the Application:** `npm start`
